2.5.3 Modifying Reporting Server

Do the following to modify the Reporting Server details:

  1. Click Manage connections on the dashboard. The Server configurations page appears.

  1. Click the Reports tab.

  2. Click the Edit icon.

  3. Modify the Report URL, if necessary.

  4. Click Update to save the changes or Cancel to discard changes.
