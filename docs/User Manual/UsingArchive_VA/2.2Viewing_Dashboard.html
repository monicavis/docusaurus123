2.2 Viewing the Dashboard

Dashboard is a visual display of all the modules associated with the Archive Viewer application. On successful login, the users are navigated to the Dashboard page.

The Dashboard has four components.

Modules​

  • Convert: Use this module to convert the CSV or Parquet files.

  • Monitor: Use this module to view the CSV, Parquet request details, utilities validation and deletion details.

  • Reports: Use this module to navigate to the Reporting page where you can view the schema details and external table details.

  • Utilities: Use this module to validate, delete, delete hold and XML check.

  • Manage users: Use this module to add, remove, edit and update the users.

  • Manage Connections: Use this module to update the details of convert server, query server, metadata database and report URL.
