2.4.5 Switching Authentication for a User

You can switch user authentication from Basic to LDAP or vice versa, if required.

Do the following to switch the user authentication:

  1. Login as an Administrator The admin dashboard appears.

  2. Click Manage users. The Users page appears.

  1. Click Switch authentication. The Switch authentication page appears.

  1. Type the IP address in the Server IP/DNS field.

  2. Type the port number in the Port field.

  3. Type the username and password in the Privileged user DN and Password fields, respectively.

  4. Select the Use TLS checkbox to connect via LDAPs, otherwise, leave it unselected.

  5. Type the DN username in the Base DN field from where the application should search for the user.

  6. Type the attribute value in the Search by username attribute field based on which the username will be validated.

  7. Click Test. On successful connection, the Success message dialog box appears.

  1. Click Done.

  2. Click Next to add a user. The Success message appears as in the below dialog box.

  1. Click Done. The super user will be logged out and the Sign in page appears.

  2. Type the newly created LDAP credentials in the Username and Password fields.

  3. Click Login. The dashboard applicable for super user appears.

  1. Click Manage users to navigate to the Users page.

  2. Click Add new. The Import user page appears.

  1. Select Administrator from the Select role drop-down field.

  2. Select the username attribute from the Select attribute drop-down field.

  3. Type the username in the Search by field and click Search to search LDAP users.

  • If you click on search, dialouge box will appear with password field for particular User

  1. Select any user and click Import user. The Import user dialog box appears.

  1. Click OK. The imported LDAP user details are updated in the Users page as shown below.
