2.7 Monitoring the Process (Admin and Operator Roles only)

Do the following to view the CSV, Parquet request details, utilities validation and deletion details:

  1. Click Monitor on the dashboard. The Monitor page appears.

  1. Click on the Request ID available in the CSV tab for which you want to view the details. The CSV Request details page appears.

  1. Click View console. The Console page appears.

  1. Click OK.

  2. Click View report. The CSV conversion report page appears.

  1. Click Download PDF to download the report or Cancel to navigate to the CSV Request details page.

  2. Navigate to the Parquet tab.

  1. Click on the Request ID for which you want to view the converted Parquet file. The Parquet request details page appears.

  1. Click the View icon under the Parquet files column of which CSV file you want to view the converted Parquet file. The View parquet file of a particular CSV file dialog box appears.

  1. Click OK.

  2. Click View console. The Console dialog box appears.

  1. Click OK.

  2. Click View report. The Parquet conversion report dialog box appears.

  1. Click Download PDF to download the report or Cancel to navigate to the Parquet request details page.
