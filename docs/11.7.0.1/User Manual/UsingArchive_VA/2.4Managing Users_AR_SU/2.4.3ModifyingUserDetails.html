2.4.3 Modifying User Details

Do the following to modify the user details:

  1. Login as an Administrator The admin dashboard appears.

  2. Click Manage users. The Users page appears.

  3. Click the Edit icon for the user you want to modify details. The Edit user page appears.

  4. Modify the fields you want to change.

  5. Click Update. The Edit the user dialog box displays message as User edited successfully.

  1. Click OK to save the changes.
