2.8.1 Validating CSV or Parquet Files

On successful conversion of the CSV or Parquet file, you can validate the file based on the application name using the Utilities > Validation tab.

Validating CSV Files​

Do the following to validate CSV files:

  1. Click Utilities on the dashboard. The Utilities page appears.

  2. Click the Validation > CSV tab.

  1. Select an application name in which you want to validate CSV files from the Application name drop-down list.

  2. Click Validate. An inline success message appears.

  1. Click the click here to view reports link. The Monitor page appears with the validation status displayed under the Utilities tab.

  1. Click the View icon under the Reports column. The CSV validation report dialog box appears.

  1. Click Download PDF to download the report or Cancel to navigate to the Monitor page.

Validating Parquet Files​

Do the following to validate Parquet files:

  1. Click Utilities on the dashboard. The Utilities page appears.

  2. Click the Validation > Parquet tab.

  1. Select an application name in which you want to validate Parquet files from the Application name drop-down list.

  2. Click Validate. An inline success message appears.

  1. Click the click here to view reports link. The Monitor page appears with the validation status displayed under the Utilities tab.

  1. Click the View icon under the Reports column. The Parquet validation report dialog box appears.

  1. Click Download PDF to download the report or Cancel to navigate to the Monitor page.
