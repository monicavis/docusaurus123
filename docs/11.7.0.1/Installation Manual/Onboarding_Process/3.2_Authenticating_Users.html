3.2 Authenticating Users

Basic authentication​

You can create a super user with basic credentials.

Do the following to create a super user with basic credentials.

  1. Type the name of a basic user in the Name field.

  2. Type the username and password of the database in the Username and Password fields respectively.

note

The password should satisfy below conditions:

− 8-20 characters

− At least one UPPERCASE character

− At least one lowercase character

− At least one Number

− No spaces

− At least one Special Character from the following:

!~@#$%^&*.-

  1. Type the same password in the Confirm password field.

  2. Type the email address.

  3. Click Next after you specify all the above details to navigate to the Convert server tab.

LDAP authentication​

You can create a super user with LDAP credentials.

Do the following to create a super user with LDAP credentials.

  1. Type the hostname or the IP address of the server in the Server IP/DNS field.

  2. Type the port number in the Port field.

  3. Type the username in the Privileged user DN field.

  4. Type the password of the user in the Password field.

  5. Select the Use TLS checkbox to connect via LDAPs, otherwise, leave it unselected.

  6. Type the DN username in the Base DN field from where the application should search for the user.

  7. Type the attribute value in the Search by username attribute field based on which the username will be validated.

  8. Click Test to authenticate users.

  9. Click Done when the connection is successful or Cancel to discard the changes.

  1. Click Next after you specify all the above details to navigate to the Convert server tab.
