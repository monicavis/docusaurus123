4.1 Accessing the Archive Viewer API

Do the following to access the Archive Viewer API:

  1. Navigate to the Swagger Page.

  2. In the Login API section, Expand POST/login and Click Try it out.

  3. Type Username and Password and click Execute.

An access token is generated if the entered username and password are correct.

An error code is displayed if the entered username and password are incorrect.

  1. Copy the access token.

  2. Click Authorize on the swagger page top-right corner.

  3. Paste the copied access token in the Value field and Click Authorize.

    Now, you can access the Archive Viewer API
