2.3.3 Modifying User Details

To modify a user, follow these steps:

  • Login as an administrator. The administrator dashboard appears.

  • Click on Manage users module

  • Click the Edit icon for the user you want to modify details. The Edit user page appears.

  • Modify the fields you want to change.

  • Click Update. The Edit the user dialog box displays message as User edited successfully.

  • Click OK to save the changes.
