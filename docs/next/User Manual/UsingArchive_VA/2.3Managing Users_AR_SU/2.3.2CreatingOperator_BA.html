2.3.2 Creating an Operator or a Business Analyst User

To create an operator or a business analyst user, follow these steps:

  • Login as an administrator. The administrator dashboard appears

  • Click on Manage users module.

  • Click Add new. The Add new user page appears.

  • Provide the name of the user in the Name field.

  • Select Operator or Business analyst from the Select role drop-down list, as needed.

  • Provide the username and email ID in the Username and Email fields, respectively.

  • Click Add user to create an operator or a business analyst user or Cancel to navigate to the Users page without creating a user.

If you click Add user, the Added user dialog box appears as shown below.

  • Click OK. The operator or business analyst user is added to the users list.
