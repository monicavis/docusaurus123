2.3.1 Creating an Administrator User

To create an administrator user, follow these steps:

  • Click Manage users on the Dashboard. The Users page appears.

  • Click Add new The Add new user page appears.

  • Provide the name of the user in the Name field.

  • Select Administrator from the role drop-down list.

  • Provide the unique username and email ID in the Username and Email fields, respectively.

  • Click Add user to create an administrator user or Cancel to navigate to the Users page without creating a user.

  • If you click Add user, the Added user dialog box appears as shown below.

  • Click OK. The administrator user is successfully added to the users list.

  • Click the Copy icon to copy the auto-generated temporary password.

  • Click the logout icon to log off from the application. The Sign in page appears.

  • Provide the newly created administrator user name in the Username field and provide the auto-generated temporary password that was copied, in the Password field.

  • Click Login, an inline message appears as “please set your password” along with fields to enter passwords.

  • Provide a new password you want in the Password and Confirm password fields.

  • Click Set password, an inline message appears as Password has been set. Now you can login with new password.

  • Click Login. The dashboard for an administrator appears.
